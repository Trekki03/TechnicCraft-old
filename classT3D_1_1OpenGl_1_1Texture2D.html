Reads a texture from a file path and creates an OpenGl texture from it.

Parameters
texturePathPath of the file which contains the image used for the texture.
inputFormatFormat in which the image is available on disk (e.g. GL_RGB or GL_RGBA)
internalFormatFormat in which the image should be internally saved (e.g. GL_RGB or GL_RGBA)
sAxisWarpModeThe warp mode of the texture on its s-axis (also named x-axis, in 2d cartesian coordinate system)
tAxisWarpModeThe warp mode of the texture on its t-axis (also named y-axis, in 2d cartesian coordinate system)
downscalingFilterThe texture downscaling function that should be used (e.g. GL_NEAREST)
upscalingFilterThe texture upscaling function that should be used (e.g. GL_NEAREST)
mipmapDefines, if mipmaps should be generated or not. If set to false, mipmaps can be generated later with GenerateMipmaps().

Usage Example

Texture2D texture("res/textures/TextureAtlas.jpg", GL_RGB, GL_RGB, GL_REPEAT, GL_REPEAT, GL_LINEAR, GL_LINEAR, false)
