Configures a Vertex Attrib pointer like ConfigureVertexAttribPointer but also enables it afterwards. A VBO has to be set via SetVertexBuffer before usage.

Parameters
indexIndex of the attribPointer
sizenumber of components per vertex attribute (must be 1, 2, 3 or 4)
typedata type of components (e.g. GL_FLOAT)
normalizedspecifies if float/double data should be normalized or used as is
stridebyte offset between to sets of components (start to start)
offsetoffset of the components to the start of the buffer (in bytes)
